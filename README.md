# DalandanAIO
[Hanbot](https://t.me/hanbot_never_die) shard, script for league of legends

Supported champions:
- Twisted fate
- Xerath
- Yasuo

Utilities: 
- Arena auto reload 
- Auto ping ward (teammates will see ping) 
- Write in chat enemy ult cooldown
